# Vuejs - Vue cli - Projesi

Vuejs Projesi

## Kurulum

### 1. Projeyi Kopyalayın
```sh
 git clone <repo-link>
 cd <proje-dizini>
```

### 2. Bağımlılıkları Yükleyin
```sh
npm install
```

### Proje Çalıştırma
```sh
npm run serve
```


## Proje Çalıştırma Sitesi
```sh
http://localhost:8080/
```

## Proje Çalıştırma Sitesi
```sh
https://vuejs-vuecli-cron-api-test.vercel.app/
```

### Build Yapma
```sh
npm run build
```

### Hata Durumu
```sh
npm run lint
```
